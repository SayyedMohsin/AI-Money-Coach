# 💸 AI Money Coach (India Focused)

AI Money Coach ek powerful aur intelligent financial management tool hai jo specially Indian users ke spendings ko track aur analyze karne ke liye banaya gaya hai. Ye app na sirf aapke kharche save karta hai balki AI ka use karke aapko bachat (savings) ki salah bhi deta hai.

## 🚀 Features
- **Smart Dashboard:** Current balance, monthly expenses, aur visual charts (Chart.js).
- **Bilingual AI Advisor:** Ask questions in English or Hinglish about your finances.
- **Monthly Roll-over:** Previous month's balance automatically carries forward.
- **Professional Invoices:** Generate and download PDF/CSV reports with itemized notes.
- **100% Privacy:** Local database management with FastAPI backend.

## 🛠 Tech Stack
- **Backend:** FastAPI (Python), SQLite
- **Frontend:** HTML5, Tailwind CSS, Chart.js, FontAwesome
- **AI Logic:** Custom Rule-based Bilingual Engine
- **Reports:** html2pdf.js, CSV export
